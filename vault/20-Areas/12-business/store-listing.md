# Store Listing Copy

> App Store / Play Store text. Source of truth: `store_listing/` в repo.

## Files

- `store_listing/short_description.txt` — one-liner (used as App Store subtitle / Play Store short description)
- `store_listing/full_description.txt` — full marketing copy

## Short description

> "AI-powered TCG card scanner: detect defects, check prices, grade condition."

## Positioning

**Audience**: serious TCG collectors (English-speaking, EU/US focus).

**Hook**: identify any Pokemon card в any language (50K+ catalog, multilingual).

**Value props** (per long copy):
1. **Instant card recognition** — multi-language OCR + AI
2. **Real-time CardMarket prices** — EU market focus
3. **AI defect detection** — surface, edges, corners, centering
4. **Professional condition reports** — grade distribution, ROI calc, shareable
5. **Collection manager** — portfolio + tracking
6. **Price watchlist & alerts** — track wanted cards

## Tone

- Confident ("ultimate", "professional-grade")
- Collector-savvy (mentions "near mint floor", "centering misalignment", "PSA grade probability")
- Action-oriented (clear "how it works" steps)

## What's missing / future

- Localized versions (currently EN only)
- Screenshots requirements / promotional images — not tracked в repo
- Keywords / ASO strategy — not documented

## When updating

- Update both files в `store_listing/` (committed to git)
- Update this note's snapshot if positioning shifts
- Cross-check с product priorities — claimed features ≠ shipping reality should be flagged
